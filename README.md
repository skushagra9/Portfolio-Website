## Getting Started 🚀
### Setup

1. Clone the repository

   ```sh
   git clone https://github.com/skushagra9/Whatsapp-AI-Bot.git
   ```

2. Install dependencies

   ```sh
   npm install
   ```

3. Run the server
    ```sh
    npm run dev
   ````
